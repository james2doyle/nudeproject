nudeproject
===========

A starting point for new web projects, meant for building simple sites or landing pages.

![NudeProject Logo](https://raw.github.com/james2doyle/nudeproject/master/nudeproject.png)

### grunt

included grunt taks

* imagemin -- compress images
* uglify -- compress js
* cssmin -- compress css
* concat -- combine files
* sass -- libsass task
* watch -- livereload
* autoprefixer -- no more prefixes

### stylesheets

* normalize.css
* grid.scss -- simple grid
* mixins.scss -- simple mixins
* basic styles for font normalization

### javascript

modernizr custom build

* cssanimations
* csscolumns
* csstransforms
* csstransforms3d
* localstorage
* websockets
* webworkers
* touch
* shiv
* cssclasses
* teststyles
* testprop
* testallprops
* prefixes
* domprefixes
* css_filters
* script_async
* script_defer
